# Demonstração de criação  de um projeto em JAVA com acesso á uma base de dados MySQL
Neste repositório serão incluídas todas as etapas para a criação de um sistema  de controle de Ordens de Serviço com sistema de login e verificação de nível de usuário logado dentro do sistema,além de prover acesso á emissão de relátorios das ordens de serviços geradas.
